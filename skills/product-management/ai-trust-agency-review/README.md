# AI Trust Agency Review Skill

This is a downloadable Cursor skill that reviews AI product ideas, PRDs, feature specs, launch plans, agent workflows, and trust claims for user agency, control, proof, recourse, displacement risk, autonomy level, failure handling, and release readiness.

It is the agent-facing companion to the essay "People Don't Fear AI. They Fear Losing Agency."

## Install

Copy this folder into a Cursor skills directory:

```txt
~/.cursor/skills/ai-trust-agency-review/
```

For a project-specific install, copy it into:

```txt
.cursor/skills/ai-trust-agency-review/
```

## Use

Ask Cursor to apply the `ai-trust-agency-review` skill to an AI product idea, PRD, feature spec, launch plan, agent workflow, or article draft.

Example prompts:

```txt
Use the AI trust agency review skill on this PRD.
```

```txt
Review this AI feature for user agency, control, proof, recourse, and release readiness.
```

```txt
Does this agent workflow deserve the level of trust the interface invites?
```
