# AI Agency Review Skill

A downloadable skill that pressure-tests an AI feature from the user's point of view: control, proof, recourse, displacement risk, autonomy level, hidden work, and whether users keep meaningful authority.

It is the agent-facing companion to the essay "People Don't Fear AI. They Fear Losing Agency."

This is one of three complementary product-management skills:

- **`ai-design-review`** — how to build the feature well (design and architecture craft).
- **`ai-agency-review`** (this skill) — whether users keep agency and will trust it.
- **`ai-stewardship-review`** — whether it should ship (governance and the release gate).

## Install

Copy this folder into a Cursor skills directory:

```txt
~/.cursor/skills/ai-agency-review/
```

For a project-specific install, copy it into:

```txt
.cursor/skills/ai-agency-review/
```

## Use

Ask Cursor to apply the `ai-agency-review` skill to an AI product idea, PRD, feature spec, agent workflow, or article draft.

Example prompts:

```txt
Use the AI agency review skill on this PRD.
```

```txt
Review this AI feature for user control, proof, recourse, and autonomy.
```

```txt
Does this agent workflow deserve the level of trust the interface invites?
```
