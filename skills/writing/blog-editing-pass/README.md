# Blog Editing Pass Skill

A downloadable skill that runs editorial passes on blog drafts and AI product essays — line edits, structural edits, and publication passes — without sanding down the author's voice.

It applies the repo's writing standards: `rules/writing.mdc` (voice and bans) and the `human-grade-writing` skill (substantive rewrites).

## Install

Copy this folder into a Cursor skills directory:

```txt
~/.cursor/skills/blog-editing-pass/
```

For a project-specific install, copy it into:

```txt
.cursor/skills/blog-editing-pass/
```

## Use

Ask Cursor for an editing pass on a draft.

Example prompts:

```txt
Do an editing pass on this draft.
```

```txt
Line edit this post — tighten it but keep my voice.
```

```txt
Publication pass: get this ready to ship.
```
