# Human-Grade Writing Skill

A downloadable Cursor skill that turns AI-assisted drafts into specific, authorial, credible prose. It improves voice, rhythm, judgment, specificity, and reader trust, and it diagnoses why a draft sounds synthetic.

The skill carries supporting layers referenced by relative path:

- `civic-prose.md` — dignity, public seriousness, moral weight.
- `rhetoric.md` — cadence, compression, persuasion, speech-like force.
- `modes.md` — format-specific defaults.
- `examples.md` — weak/strong rewrite examples.

## Install

Copy this folder into a Cursor skills directory:

```txt
~/.cursor/skills/human-grade-writing/
```

For a project-specific install, copy it into:

```txt
.cursor/skills/human-grade-writing/
```

## Use

Ask Cursor to humanize or sharpen a draft.

Example prompts:

```txt
Make this sound like a person with judgment wrote it.
```

```txt
Remove the AI voice from this post.
```

```txt
Diagnose why this draft sounds synthetic, then rewrite it.
```
